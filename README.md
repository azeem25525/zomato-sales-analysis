# Zomato Sales Analysis – Power BI

## Objective
Analyze Zomato orders and revenue to identify top cities, restaurants, cuisines, and peak order trends.

## Tools
Power BI, DAX, Excel (SQL optional)

## KPIs
- Total Revenue
- Total Orders
- Average Order Value (AOV)
- Orders by City
- Top Restaurants by Revenue
- Cuisine Performance
- Peak Order Time (Hour/Day)
